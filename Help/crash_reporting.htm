Crash Reporting

Unimplemented - SMTP code written, but does not send reports. This feature is subject to change.

The crash reporting feature of Magenta allows you to send, upon abnormal shutdown of the program, a report detailing the current state of the program to the program author.

To use this feature, check off Enable automatic crash reporting and fill in the fields. Return e-mail address is your address, and the SMTP server is found by checking your email settings, if you have a POP3 mailbox. If you don't, you cannot use the crash reporting feature.
