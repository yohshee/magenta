Block List

The Block List feature of Magenta is a way to ignore people that you know that are going to cause problems without having to ignore them each time you enter the chat room. Unlike the standard Ignore feature, however, the Block List operates on name recognition, rather than mangled IP address recognition, and is entirely maintained by Magenta rather than the chat server.

Also, instead of vanishing entirely from the Who is Online list, people in the Block List appear embellished with the string "[Blocked]" after their name.

Access your Block List by going to Chat -> Block List. To add a name, type it in the Name to add text box, and click Add Name. To remove one from the list, simply select it from the list, and click Remove Name.

To save your changes to the block list and close the dialog box, click Save and Close. To close the dialog box without saving, click Discard Changes.
