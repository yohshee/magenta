Ignore Options

In the Other Options area of the Options dialog box, there is a set of options dealing with the Ignore feature of Magenta. It is highly recommended that you LEAVE THEM ALONE, except for Display a message when someone ignores you. Turning that one off is perfectly all right.

The Mutual Ignore option controls whether or not Magenta automatically ignores the other party when someone ignores you. Leave it checked.

Send the ignore command when ignoring controls whether or not the Ignore command is actually sent to the server when you ignore somebody. If you uncheck this, they will not mutually ignore you when you ignore them. Generally an undesirable option. Leave it checked.
