Private Chat

Private chat utilizing the main chat window is very similar to regular chatting in nearly every way; every option that was mentioned in the public chat section applies to private chat when using the main window. The only difference is that you select a user from either list box and click Send Private to send your message.

It's when you start using the private chat windows that things get interesting.

To open a private chat window, select a user from either list box, and click the Private Chat button. Any private message from this person will now be dispatched to this window.

To send messages, type your message in the bottom text box and hit Enter, or click the Send button. All private chat window messages are timestamped as they appear.

Closing the chat session is done by simply clicking Close. You can save a plain-text log of the session, as well, by clicking Save Log.
