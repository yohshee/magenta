Getting Started

Getting started is very easy:

  1. Unzip the Magenta program file using a zip utility such as WinZIP or WinRAR.
  2. Place it anywhere you want. Make a folder just for Magenta, if you want to. (Recommended if you're reading this help file)
  3. Double-click on the Magenta executable.
  4. Enter your name and start chatting.

The Magenta interface behaves just like that of the Cyan Chat Java client (with some added enhancements), so the transition from that client to Magenta is normally as smooth as silk. And it takes less time to start up, too.
