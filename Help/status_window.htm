Status Window

The Magenta Status Window, accessed by using Tools -> Status Window, is just another way of looking at your user lists.

Cyan employees, Cyan guests, and regular chatters are all shown in this window, and are discerned by differences in their icon appearance (e.g. the Cyan employee icons all wear cyan-colored shirts). Hovering your mouse pointer over an icon will give you a little information about the user, in particular their mangled IP address.

You can change how the list view is shown by going to the View menu, and selecting either Large Icons or Report. The report view will display the mangled IP address, along with the user name.

The File menu allows you to either Close the status window, or Save a file containing all of the users and their associated information. This file has an extension of .log, and is formatted using the Windows INI format.

Selecting Log All Incoming Packets will prompt you for a filename, and, following that, it will log all usernames every time the Chat Server sends a who list update.
