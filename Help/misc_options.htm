Miscellaneous Options

There are many other options that do not fall under any other particular category. The Auto Logging button is discussed in the Logging Chats section.

Show Chat Console at startup - forces Magenta to show the chat console once Magenta starts.

Block the Chat Server - Keeps Magenta from showing ChatServer messages. Handy if you're stuck on the CC debug server (1813) because the main one is down.

Notify about events when minimized - Magenta will flash its button on the taskbar if a message arrives if this option is set. Also, if the system tray icon is visible, it will flash the icon.

Open URLs in a new window - Magenta will go through the traces of opening all URLs in a fresh new browser window if you check this off.

Hide notification icon - Hides the notification area icon if checked.

Name Color button - Clicking this button will bring up a standard Windows color dialog. Select one of the colors and click OK. Once you do, your chat name will always show up in that color in the header of each message.
