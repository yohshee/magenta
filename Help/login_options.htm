Startup Options

All startup options are accessed by going to Tools -> Options.

Filling in the Default Name textbox allows you to set your name at startup; Magenta automatically saves your name for you if you click OK in the Options dialog. If you check off Log into chat at startup, then Magenta will make an attempt to log into Cyan Chat if you filled in your username once the application starts.

Host Information allows you to change where Magenta connects. You'll rarely change the default options in this area, unless you desire to use the CC debug server that MarkD maintains for Cyan Chat developers. This server is located at cho.cyan.com:1813. The standard CC server is located at cho.cyan.com:1812, and is always the default server for Magenta. If you check off Connect to Cyan Chat at startup in the Other Options area, Magenta will automatically attempt to connect to this server when you start the program. This option is selected by default, to keep with the behavior of the Cyan Chat Java client.
