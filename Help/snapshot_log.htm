Snapshot Logs

Snapshot logs are a way to take a quick "picture" of the current state of the chat buffer and save it to a file.

In other words, it just grabs all of the text you've received so far, without making a single change to it, and stuffs it into a file. No biggy. Saving the file as Rich Text will preserve any formatting, such as color.

To save a snapshot log, all you have to do is select Chat -> Save Snapshot Log, select whether you want to save it as Rich Text Format (RTF) or text (saving it as text allows you to choose either the extension .log or .txt), and enter the filename. Simple.
