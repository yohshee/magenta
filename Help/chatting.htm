Chatting

There's not much different between chatting with Magenta and chatting with the regular Cyan Chat client.

You follow the same steps regardless of which client you use:

  1. Start the chat client.
  2. Establish a connection to the chat server (this is always done automatically by the Java client, and by default done automatically by Magenta)
  3. Choose a name for yourself, minding the rules. Both clients enforce them.
  4. Click the Join Chat button, and start chatting.
  5. Once you're finished chatting, click the Leave Chat button.

Both clients allow you to send and receive private messages, as well, but Magenta gives you the option of using private message windows to isolate private messages from the main chatting area, and to help you stay a bit more organized when chatting.

Also, there are some fan-maintained CC servers out there and a special chat server, and in your preferences, you can change where you connect. Connecting to a server is easy, once the settings are configured: just go to Chat -> Connect. If there are any errors, they will be displayed in the status bar. Disconnecting is done by selecting Disconnect in that same menu.
